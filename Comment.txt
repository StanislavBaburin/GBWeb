float - обтекание текста
LoremКолвоСимволов - добавить тест
Текст обтекает блоки со свойством флоат, но блочные элементы их не видят и "наезжают" на них
Чтобы не наезжали, нужно у нижнего элемента прописать clear:both, тогда блочные элементы с флоат станут значимыми.

Если у двух блочных элементов есть вертикальные отступы, они не суммируются, а выбирается наибольший, но если у нижнего блока будет свойство флоат лефт, то отступы суммируются

Если элементы накладываются и надо убрать наложение, но оставить расположение, можно использовать свойство overflow:hidden;(скрыть наложение)

Чтобы избежать схлопывание нужно использовать float:left;


Position
	relative
блок не изменяет положение других элементов на странице, при применении этого свойства появляются свойства top, left и тд для изменения положения на странице. При этом остальные блоки будут считать что этот блок находится на своём положенном месте и занимает столько пространства сколько обычно, даже если его сместить
z-index - кто впереди на экране
	absolute
позиционировние от левого верхнего угла
Блок выпадает из структуры, остальные его не видят(например звонок(позвоните нам))
 
Центрирование - у блока указать margin:0 auto; 

Смешанный relative-absolute
Создаем блок с классом container
туда записываем другие блоки
У родителя position:relative;
Если у ребенка указать absolute, то точкой отсчета будет левый верхний угол родителя
Чаще всего применяется именно он так как у absolute при разном разрешении блоки могут смещаться, а у relative занимается то место на странице, где эти блоки были изначально

Обычная структура в бади:
основной контейнер
в нем хэдер, мэйн и футер
остальное уже распихиваем по блокам либо добавляем ещё

Для контейнера указать макс и мин значение ширины
Указывать для шапки мин высоту
Убирать паддинги и макрджины для всех в начале

Как прижать футер в нижнюю часть экрана
в цсс для хтмл и бади задать высоту 100%
для контейнера задать мин.высоту 100%
Затем футер вынести за блок-контейнер
затем для футера задаем марджин топ -200
Чтобы текст из мейн не наползал на футер, можно перед футером создать пустой блок с такой же высотой


bootstrap - фреймворк для создания сайтов как в конструкторе, по шаблону
1.Cкачиваем
2.Перекидываем в папку проекта скачанные файлы/папки
3.Создаем index.html и копируем в него с getbootstrap.com код Basic template
