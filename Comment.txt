	Объявление in line

чтобы редактировать сразу несколько строк, нужно выбрать необходимые месте с зажатым ctrl 
+:быстро редактируется элемент
-:указывать каждому элементу

используются в основном в письмах

	Стили в разделе Head
внутри head добавляем тег <style></style>
внутри <style></style> например пишем 
h3 {
color:red;
}
Стиль используется для всей страницы, на который мы пишем

	Подключение внешнего файла
Создать файл style.css в одной директории с index.html
Подключить внутри тега head с помощью <link rel="stylesheet" href="">
в style.css пишем например
h3 {
color:red;
}

Селектор может быть тегом, идентификатором, классом, атрибутом.

id уникален, классов может быть несколько
в CSS идентификатор указывается после #, класс после .

В Brackets можно нажать на класс и нажать сочетание Ctrl+e, появится окно в котором можно редактировать выбранный класс

обращение к атрибуту:
тег[атрибут="значение"]{свойство:значение;}

Чтобы обратиться к вложенным элементом, родителю присваиваем класс, а в css снчала указываем класс а через пробел тег, для которого указываем свойства
так находятся все вложенные теги, если вместо пробела поставить знак >, то будет применяться только к дочерним(первым дочерним а не вложенным в них)

можно обратиться к классу и соседним тегам, для этого указываем класс потом + потом тег и данные свойства применятся только для соседа
